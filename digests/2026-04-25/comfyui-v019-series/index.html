COMFYUI RELEASES ↗ 2026-04-13〜17

ComfyUI v0.19 系が SeeDance 2.0 / Ace Step 1.5 XL / Ernie Image / WAN2.7 を一気に追加、動画・画像・音声のモデル対応が大幅拡張

4日間で4バージョン連続リリース。 特に v0.19.0 は 65+ PRs の大型更新で、SeeDance 2.0 / Ace Step 1.5 XL / Ernie Image / Qwen3.5 / WAN2.7 / LTX2 / Tencent3D などのノードが一気に追加された。 その後 v0.19.1〜.3 で連続バグ修正・小機能追加。

TL;DR
ComfyUI v0.19 系統で、動画生成（SeeDance 2.0 / LTX2 / WAN2.7）・画像生成 （Ernie Image / LongCat-Image / Ace Step 1.5 XL）・音声生成（Sonilo Audio）・3D（Tencent3D / Hunyuan3D）のパートナーノードが大量追加。Intel XPU スタンドアロンパッケージや量子化モデルの OOM バグ修正も同期間で入った。
